# Browser New Tab Dashboard

A minimal custom new tab dashboard for Chrome and Edge.

Features

- Clock
- Web search
- Quick links
- Clean dark UI

## Install

Chrome

1. Go to chrome://extensions
2. Enable Developer Mode
3. Click "Load unpacked"
4. Select this folder

Edge

1. Go to edge://extensions
2. Enable Developer Mode
3. Click "Load unpacked"
4. Select this folder
